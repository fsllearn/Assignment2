  instruction to create git Repository
"""
1) github.com, create Repository
- copy the URL link
2) in your laptop, run the mingw64.
3) create sub-directory e.g.  Assignment2

- git clone "<URL>"
- create file e.g. README.md and Assignment2.py
- git Add
- git commit
- git push
